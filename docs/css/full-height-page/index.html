Create a Full Height Page

NOTE: This page doesn't have base styles applied yet so things look kinda busted.

Given an HTML structure like

<body>
  <header></header>
  <main></main>
  <footer></footer>
</body>

This CSS will space things out so the `header` is at the top of the page and the page and the `footer` is at the bottom with the `main` taking up all the other space. 

* {
  margin: 0;
}

html {
  scrollbar-gutter: stable;
}

body {
  min-height: 100svh;
  display: grid;
  grid-template-rows: auto 1fr auto;
}

This is not a sticky header. It just makes sure the page takes up at least the full height of the viewport. If the page is longer the footer will be pushed down below the viewport as normal.

The grid does most of the heavy lifting. The `scrollbar-gutter: stable` keeps short pages and pages that have a scrollbar working together nicely instead of shifting over when going from a page with a scrollbar to one without one.

I got this technique from Kevin Powell's video 3 modern CSS features to add to your reset.
