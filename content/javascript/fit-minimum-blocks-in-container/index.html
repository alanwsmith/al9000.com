Fit Blocks in a Container with a Minimum Number of Columns

[@ date(p.created) @]
Expand for Details
  • This code makes divs that fill a container element (it's the prototype for the multi-player).
  • The number of rows and columns is determined automatically based on the width and height of the containing element.
  • A minimum of five columns are output.
  • A maximum width is defined for the div cells. More columns are added if two can be added without increasing beyond the max.
  • Columns are added in pairs so there's always an even number.
  • A width/height ratio is used to determine how tall the individual div cells are.
