Split a Files Bytes

[!- endblock -!] [!- block main -!] [! filter md !] This started as an idea for how to split things up so that fair use files don't get pinged by automatic bots. The idea being to split a file into two with every other byte alternating between the outputs. Reassembly to be done with WASM. Then, I realized I could just encrypt the files. They could be decoded automatically by simply including the password in the javascript on the page. Or, you could set up a form so that only folks with the password could get them. I really like that idea in general. Basically, a way to serve encrypted content on a static site. See: TKTKTK for an example [! endfilter !] [!- endblock -!]
