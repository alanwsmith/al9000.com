Welcome to al9000.com

My main site is alanwsmith.com. It's been around for ~25 years. It's been great, but I like the idea of a site that's not my name. So, I'm making this one.

I've made a ton of sites little sites that are under the alanwsmith.com domain. A big reason for that is because the tools I use to build the main site weren't well set up to make independent pages. I'm writing new software to make that easier to do here. It'll make maintaining and navigating between different experiments a lot easier.

The idea is slowly but surely move everything here over time. New stuff will start here as well. So far, the only thing here is a work-in-progress Magic Deck Builder.

It'll take a while for there to be much here. Stop back by every now and then to see what's up.
