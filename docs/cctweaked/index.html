CC:Tweaked (aka ComputerCraft)

  • This is where I'm moving all my ComputerCraft/CC-Tweaked lua scripts for Minecraft.

  • NOTE: Most of the lua scripts are in place, but they don't currently deploy to the site because they start with a _ which prevents them from moving in the current iteration of the site builder.

  • TODO: You can load everything by calling

    wget run https://al9000.com/cctweaked/init.lua

    That runs an initialization script the sets up a computer/turtle.

  • TODO: The local-scripts folder contains scripts that are copied directly to the computer/turtle.

  • TODO: The remote-scripts folder contains scripts that are called remotely each time they are run so I can edit them in a text editor outside of Minecraft itself.

  • NOTE: As of March 2026 version 26.x of Minecraft is out but CC:Tweaked only runs on 1.21 or earlier. We'll see if it gets updated.

  • Modrinth local file storage:

    ~/Library/Application Support/ModrinthApp/profiles/PROFILE_NAME/saves/WORLD_NAME/computercraft/computer/COMPUTER_NAME

  • Turtles have a max fuel level of 100,000.

  • Advanced Turtles have a max fuel level of 200,000.
